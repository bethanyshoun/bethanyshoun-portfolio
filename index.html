About

Bethany thrives on solving problems through innovation and attention to detail. Developing web applications that are responsive, accessible, and creatively designed to streamline time-consuming tasks for peoples' everyday lives are her key interests.

Projects

Rent-A-Party


An ecommerce site with React front-end, utilizing Mongo DB that allows users to select items to rent for parties, add them to a cart, and pay. This collaborative group assignment was the cummulinating project for our coding bootcamp.

Word Raffle


Music-lovers show off their knowledge of song lyrics when given a random word by posting song lyrics containing that word.

Cat Genie


For cat enthusisats and those learning to love cats- immerse yourself in the world of felines through funny images and gifs + random facts about cats. In this collaborative group project using the Bulma framework, facts and images are fetched from server-side APIs. Users can save a favorite fact and a favorite image in a persistent local storage container.
